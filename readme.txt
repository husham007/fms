


File management application developed in:

Node.js, Express.js
MongoDB (Database)
Jade/Pug (Template Engine for html pages)
jQuery for client side delete operation
Bootstrap/css for client side

app.js (main application file)
models (folder, that keeps database schema)
views (folder, that keeps webpages)
routes (folder, that keeps request, response functions for each request for users and files (articles))
config (folder, that keep strategies for both type of users, admin and basic user)

databse link is database:'mongodb://localhost:27017/nodekb

Please note that name articles or article refers to files or file
